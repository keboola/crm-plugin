# Keboola CRM — Claude Code plugin

Public distribution of the `keboola-crm` Claude Code plugin. The
source of truth is Keboola's private CRM monorepo; **this repo is
regenerated automatically on every CRM release** (currently
v0.52.1) — do not edit it by hand.

## Install

```
/plugin marketplace add keboola/crm-plugin
/plugin install keboola-crm
```

## Update

```
/plugin marketplace update
/plugin update keboola-crm
```

The plugin drives the `crm` CLI — install and configure it first
(see the plugin's own README under `keboola-crm/`).
